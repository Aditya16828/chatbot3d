# Chatbot3d
Tech Stack used:
- **Front-end**
  - HTML, Javascript
  - React.js
  - Three.js
  - Vite
- **Back-end**
  - Node.js
- **Chatbot model**
  - Dialogflow: returns the text as well as voice responses for a user query.
- **3D model**
  -[Ready player me](https://readyplayer.me/)
